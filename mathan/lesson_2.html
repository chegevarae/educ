ДЗ Урок 2. Множество. Последовательность¶

Тема “Введение в математических анализ”¶

Как относятся друг к другу множество и последовательность? (в ответе использовать слова типа: часть, целое, общее, частное, родитель, дочерний субъект и т.д.)¶

Последовательность это выборка элементов из множества объектов объединенных по одному признаку, она может быть равна по количеству элементов множеству или быть его частью, образуя таким образом подмножество. У последовательности есть свое множество. Оно так и называется - множество значений. Оно может быть конечным, может быть бесконечным, но это не сама последовательность.

Прочитать высказывания математической логики, построить отрицания и установить истинность.¶

1. Любой элемент из множества от 0 до 1 больше нуля, где сигнум = 1.
2. Любое натуральное n больше 2, где существуют натуральные элементы x,y,z, для которых справедливо равенство x в степени n равно сумме y в степени n и z в степени n.
3. Любой элемент x из существующего множества вещественных чисел X является вещественным числом, для которого справедливо X > x.
4. Любой элемент x из несуществующего комплексного y является комплексным числом, для которого справедливо условие, где x > y или x < y.
5. Любой элемент из множества от 0 до пи/2 существующего эпсилон больше нуля, для которого справедливо неравенство синус y меньше синуса суммы y и эпсилон.
6. Любой элемент из существующего эпсилон множества в пределе от 0 до пи больше нуля, для которого справедливо неравенство косинус y больше косинуса суммы y и эпсилон.
7. x существует, когда x не входит ни в какое числовое множество.
